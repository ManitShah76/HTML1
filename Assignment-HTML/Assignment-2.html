Canvas and SVG

SVG gives better performance with smaller number of objects or larger surface. SVG is a language for describing 2D graphics in XML. SVG is XML based, which means that every element is available within the SVG DOM. SVG can be modified through script and CSS.




Canvas draws 2D graphics, on the fly (with a JavaScript). Canvas has several methods for drawing paths, boxes, circles, text, and adding images.
